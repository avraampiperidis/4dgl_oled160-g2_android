Android notifications to mOled-160-G2 via bluetooth.

A very simple step by step guide/tutorial to create a wireless(bluetooth) 1.7 inch monitor, an android app to connect to it,and automatically send every notifications from android to monitor.

required components

1)uOLED-160G2 Display Module display module
2)uUSB-PA5-II Programming Adaptor adapter
3)Bluetooth module
4)Programming Cable
5)USB to 4-pin cable
6)4-pin to 4-pin cable
7)Any USB power bank
8)Workshop4 IDE link
9)Android Studio

Starting from android studio

I will post small snippets of the actual code. Clone this git to get the whole project.

first some important stuff about AndroidManifest.xml

 <uses-permission android:name="android.permission.READ_PHONE_STATE" />
    <uses-permission
        android:name="android.permission.BLUETOOTH" />
    <uses-permission
android:name="android.permission.BLUETOOTH_ADMIN" />

android.permission.READ_PHONE_STATE to get the carrier network android.permission.BLUETOOTH_ADMIN to get access to turn off n on the bluetooth

 <service android:name="com.example.abraham.iNotify.NLService"
            android:label=".NLService"
            android:permission="android.permission.BIND_NOTIFICATION_LISTENER_SERVICE">
            <intent-filter>
                <action android:name="android.service.notification.NotificationListenerService" />
            </intent-filter>
</service>

We declare our notification service class. This service will run in the background,and will receive every notification.
